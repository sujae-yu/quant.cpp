# TurboQuant.cpp

TurboQuant 논문(ICLR 2026)의 KV 캐시 압축을 순수 C로 구현한 LLM 추론 엔진입니다.

Randomized Hadamard Transform + 부호 해싱으로 key를 1비트로 압축합니다. Value는 Q4 또는 Q2로 독립 양자화. GGUF 직접 로딩과 MoE 아키텍처를 지원합니다.

Qwen3.5-35B-A3B MoE, 16GB Mac: 18.6x KV 압축으로 131K 컨텍스트. Gemma 3 4B perplexity: 4.9x 총 K+V 압축에서 +0.03%. 1-bit attention 코사인 = 0.634, 이론 한계 2/pi와 일치.

31개 테스트, ASan 클린, 외부 의존성 없음.
