# TurboQuant.cpp

Pure C implementation of the TurboQuant paper (ICLR 2026) for KV cache compression in LLM inference.

Key vectors compressed to 1 bit via randomized Hadamard transform + sign hashing. Values independently quantized to Q4 or Q2. Supports GGUF direct loading and MoE architectures.

Qwen3.5-35B-A3B MoE on 16GB Mac: 131K context with 18.6x KV compression. Gemma 3 4B perplexity: +0.03% with 4.9x total K+V compression. 1-bit attention cosine = 0.634, matching the 2/pi theoretical limit.

31 test suites, ASan clean, no external dependencies.
